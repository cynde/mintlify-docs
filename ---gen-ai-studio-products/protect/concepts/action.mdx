# Action

Actions are user-defined actions that are taken as a result of the [ruleset](/galileo/gen-ai-studio-products/galileo-protect/concepts/ruleset) being triggered. Galileo will provide a set of action types (override, passthrough), that the user can use, along with a configuration for each action type.
